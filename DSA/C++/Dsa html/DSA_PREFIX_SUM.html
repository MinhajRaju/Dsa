📖 Prefix Sum — কী, কেন, কখন?
💡 মূল ধারণা

Prefix Sum মানে হলো index 0 থেকে i পর্যন্ত সব উপাদানের যোগফল একটি array তে আগে থেকে store করে রাখা।


Key formula: pre[i] = pre[i-1] + arr[i]


Range query: sum(l,r) = pre[r] - pre[l-1]


এতে প্রতিটি range sum O(n) এর বদলে O(1) এ বের করা যায়।

🎯 কখন ব্যবহার করবো?
  • Range sum query বারবার করতে হলে
  • Subarray sum যাচাই করতে হলে
  • Equilibrium / Pivot point খুঁজতে হলে
  • Multiple queries efficiently answer করতে
  • 2D matrix এর rectangular region sum
  • Left/Right prefix এর combination দরকার হলে
🔍 এই signal দেখলে Prefix ভাবো
📌 "subarray sum" বারবার জানতে হবে
📌 "range [l,r] এর যোগফল" → prefix query
📌 "left sum == right sum" → pivot/equilibrium
📌 "product except self" → left×right prefix
📌 "2D matrix এর region sum" → 2D prefix
📌 "precompute করো, তারপর query" → prefix

Range Sum Query

② Left×Right
L[0]=1
for i in 1..n:
  L[i]=L[i-1]*a[i-1]
R[n-1]=1
for i in n-2..0:
  R[i]=R[i+1]*a[i+1]
res[i]=L[i]*R[i]

📖 Prefix + HashMap — কী, কেন, কখন?
💡 মূল ধারণা

Prefix Sum + HashMap মানে prefix sum এর সাথে একটি hash map রেখে আগের কোন prefix sum দেখা গেছে তা track করা।


Key insight: sum(l,r) = pre[r] - pre[l-1]


তাই pre[r] - target = pre[l-1] — map এ এটা আছে কিনা O(1) এ check করো।


Template: map[0]=1 দিয়ে শুরু করো (empty subarray handle করতে)।

🎯 কখন ব্যবহার করবো?
  • Count subarrays with sum = k → prefix+map count
  • Longest subarray with sum = k → prefix+map index
  • Divisible by k → prefix%k → map
  • Equal 0s and 1s → 0→-1, then sum=0
  • Subarray sum check → seen before?
  • যখন brute force O(n²) কে O(n) করতে চাই
🔍 এই signal দেখলে Prefix+Map ভাবো
📌 "count subarrays with sum = k"
📌 "longest subarray with given sum"
📌 "sum divisible by k" → prefix mod
📌 "equal 0s and 1s" in binary array
📌 O(n²) brute force কে O(n) করতে
📌 "exists subarray" with some property
⚡ তিনটি Main Pattern
① Count Sum=K
mp={0:1}, pre=0
ans=0
for v in arr:
  pre += v
  ans += mp[pre-k]
  mp[pre]++
return ans

Subarray Sum = K

② Longest Sum=0
mp={0:-1}, pre=0
ans=0
for i,v in arr:
  pre += v
  if pre in mp:
    ans=max(ans,i-mp[pre])
  else: mp[pre]=i

Longest 0-sum subarray

③ Mod Div K
mp={0:1}, pre=0
ans=0
for v in arr:
  pre=(pre+v)%k
  ans += mp[pre]
  mp[pre]++
return ans
