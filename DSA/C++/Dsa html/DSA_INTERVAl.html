📖 Array Range — কী, কেন, কখন?
💡 মূল ধারণা

Range / Interval মানে [start, end] — একটি শুরু এবং শেষ বিন্দু। Sorted array তে consecutive numbers গুলো একটা range তৈরি করে।


Key insight: chain কখন ভাঙে? যদি পরের সংখ্যাটি বর্তমানের চেয়ে ১ বেশি না হয়, তাহলে নতুন range শুরু।


Interval problems এ সাধারণত sort by start করে তারপর overlapping check করো।

🎯 কখন ব্যবহার করবো?
  • Consecutive numbers গ্রুপ করতে → Summary Ranges
  • Overlapping intervals merge করতে → Merge Intervals
  • Greedy reach — Jump Game pattern
  • Mark & Sweep — Find missing/disappeared
  • Prefix Product — left/right pass
  • Array এর structure/pattern বের করতে হলে
🔍 এই signal দেখলে Range ভাবো
📌 "sorted array" তে consecutive groups
📌 "merge/combine overlapping intervals"
📌 "missing number" in range [1..n]
📌 "can you reach end?" → greedy
📌 "compress" → write pointer + count
📌 Interval এ নতুন point insert করতে হবে

📖 Prefix Sum & Sliding Window — কী, কেন, কখন?
💡 Prefix Sum মূল ধারণা

Prefix array তে pre[i] = arr[0]+arr[1]+...+arr[i]। যেকোনো subarray [l..r] এর sum = pre[r] - pre[l-1]।


O(n) preprocessing → O(1) range query। Nested loop এর O(n²) → O(n) এ নামায়।

💡 Sliding Window মূল ধারণা

দুটো pointer l, r। r বাড়িয়ে window বড় করো, l বাড়িয়ে ছোট করো। Fixed size বা variable size window।


Key: window এ কী track করছি? Sum? Distinct count? Max? — এর উপর l কখন বাড়াবো তা নির্ভর করে।

🔍 Signal দেখলে এই pattern
📌 "subarray sum = K" → prefix+map
📌 "max sum of size k" → fixed window
📌 "longest subarray with..." → variable
📌 "range sum query" → prefix array
📌 "at most K distinct" → sliding
📌 O(n²) nested → O(n) sliding করো
⚡ Pattern Templates
① Fixed Window
sum=sum(arr[:k])
for i in k..n:
  sum+=arr[i]
  sum-=arr[i-k]
  best=max(best,sum)

Max avg, fixed K

② Variable Window
l=0, cnt={}
for r in arr:
  add arr[r]
  while invalid:
    remove arr[l]; l++
  best=max(r-l+1)

Longest valid

③ Prefix+Map
mp={0:1}, pre=0
for v in arr:
  pre += v
  ans+=mp[pre-k]
  mp[pre]++

Subarray sum=K

Easy— basic prefix/window
Medium— variable window + map
Hard— complex window logic
